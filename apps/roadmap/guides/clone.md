# Clone a Repo

Cloning copies a repo from GitHub to your computer so you can work on it locally. It's smarter than a regular download -- it keeps the connection to GitHub so you can push your changes back when you're ready.

## How to do it

1. Go to any repo on GitHub.com
2. Click the green **Code** button
3. Copy the URL shown (HTTPS is simplest)
4. Open GitHub Desktop and click **File** > **Clone Repository**
5. Paste the URL and choose a local folder
6. Click **Clone**

If you prefer the terminal:

1. Open your terminal
2. Navigate to where you want the project
3. Run the clone command

## Try it

```
git clone https://github.com/YOUR-USERNAME/my-first-repo.git
```

## Tips

- You can clone anyone's public repo, not just your own. This is how you get copies of open-source projects.
- Only clone into an empty folder or let Git create the folder for you. Cloning into an existing project folder causes problems.
