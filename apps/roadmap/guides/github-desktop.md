# Try GitHub Desktop

GitHub Desktop is a visual app for managing your repos without using the terminal. You can see your changes, make commits, push code, and switch branches -- all with buttons instead of typed commands. It's a great way to get comfortable with the workflow.

## How to do it

1. Download GitHub Desktop from [desktop.github.com](https://desktop.github.com/)
2. Install and open it
3. Sign in with your GitHub account
4. Click **Clone a Repository from the Internet**
5. Select one of your repos and choose where to save it on your computer
6. Click **Clone**

## Try it

```
Clone your repo using GitHub Desktop. Open a file in any text editor, make a change, then switch back to GitHub Desktop. You'll see the change highlighted. Write a commit message and click "Commit to main."
```

## Tips

- GitHub Desktop shows a color-coded view of your changes: green lines are additions, red lines are deletions.
- You can use GitHub Desktop alongside the terminal or any other tool. They all work with the same repo.
