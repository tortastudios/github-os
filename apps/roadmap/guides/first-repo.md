# Create Your First Repo

A repository (repo) is a project folder with memory. Every change you make is tracked, so you can always go back to an earlier version. Think of it as Google Docs version history, but for everything in the folder.

## How to do it

1. Click the **+** button in GitHub's top-right navigation bar
2. Select **New repository**
3. Name it `my-first-repo`
4. Add a short description (one sentence about what it's for)
5. Check **Add a README file**
6. Click **Create repository**

## Try it

```
Create a repo called "my-first-repo" with the README option checked. You should see your new repo page with the README displayed automatically.
```

## Tips

- Always initialize with a README. An empty repo is harder to work with.
- Repo names should be lowercase with hyphens instead of spaces: `my-project` not `My Project`.
- Public repos are free and visible to everyone. Private repos are also free but only you can see them.
