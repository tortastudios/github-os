# Install an AI Tool

AI coding tools read your repo (including CLAUDE.md) and help you work faster. They can write code, answer questions about your project, create files, and automate repetitive tasks. Pick one and start using it with your GitHub OS repo.

## How to do it

1. Choose an AI tool:
   - **Claude Code** -- runs in the terminal, reads your whole repo
   - **Cursor** -- a code editor with AI built in
   - **Claude Desktop** -- a desktop app you can point at files
   - **GitHub Copilot** -- built into VS Code and other editors
2. Install it from the tool's website
3. Open your GitHub OS repo in the tool
4. Ask it a question about your project to test it

## Try it

```
Open your repo in your AI tool and ask:

What's in this repo and how is it organized?

Then try something practical:

Help me write a README for this project. My business is [describe your business in one sentence].
```

## Tips

- Your CLAUDE.md file gives AI tools context about your project. The better that file is, the better the AI's responses will be.
- Start by asking questions about your existing repo before asking the AI to create new things. This builds trust and helps you understand what the tool can see.
