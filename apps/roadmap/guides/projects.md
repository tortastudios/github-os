# Set Up a Project Board

GitHub Projects gives you a kanban board (like Trello) connected directly to your issues. Drag tasks between columns like "To Do", "In Progress", and "Done". It's free, built into GitHub, and stays in sync with your issues automatically.

## How to do it

1. Go to your repo and click the **Projects** tab
2. Click **New project**
3. Choose **Board** as the layout
4. Give it a name (like "Q1 Tasks" or "Website Launch")
5. Click **Create**
6. Add your existing issues by clicking **+ Add item** and searching for them

## Try it

```
Create a project board with three columns: To Do, In Progress, and Done. Add your existing issues to the "To Do" column. Drag one to "In Progress" to try the workflow.
```

## Tips

- Projects can pull issues from multiple repos, which is useful if your work spans several projects.
- Use the **Table** view for a spreadsheet-like layout, or **Board** for kanban-style. You can switch anytime.
- When you close an issue, it automatically moves to "Done" on your board if you set up the built-in automation.
