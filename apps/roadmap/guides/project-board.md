# Set Up a Project Board

GitHub Projects gives you a kanban board connected to your issues. Drag tasks between To Do, In Progress, and Done. It's free Trello, built into GitHub, and stays in sync with your issues automatically.

## How to do it

1. Go to your repo's **Projects** tab
2. Click **New project**
3. Choose **Board** as the layout
4. Give it a name (like "Q1 Tasks" or "This Month")
5. Click **Create**
6. Add your existing issues by clicking **+ Add item**

## Try it

```
Create a project board with three columns: To Do, In Progress, and Done. Add your existing issues to it. Drag one to "In Progress" to try the workflow.
```

## Tips

- Projects can pull issues from multiple repos, useful if your work spans several projects.
- Use the **Table** view for a spreadsheet-like layout, or **Board** for kanban. You can switch anytime.
- When you close an issue, it automatically moves to "Done" if you enable the built-in automation.
