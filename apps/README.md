# Apps

Each project gets its own folder here. A project could be a website, a tool, an API, a game, or anything you build and ship.

## Structure

```
apps/
├── roadmap/        # Pre-installed: your progress dashboard
├── my-site/        # Example: a personal website
└── invoice-tool/   # Example: a side project
```

## Starting a New App

Create a folder, add a README, start building. Or ask AI: "help me set up a new project in apps/"
